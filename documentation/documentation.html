Dokumentasi Instalasi & Setting

Demo

Untuk hasil akhir bisa diakses disini

Spesifikasi

  • Laravel Version : 8.75
  • Database : MySQL
  • Library : jQuery, DataTable
  • Template : Sistla
  • Fitur :
    1. Login
    2. Master Barang
    3. Transaksi Barang Masuk
    4. Transaksi Barang Keluar
    5. Report Stock
    6. Manajemen data stock barang menggunakan TRIGGER
    7. Input detail barang masuk dan barang keluar menggunakan AJAX
  • Kekurangan :
    1. Input master barang belum ada validasi untuk duplikasi kode barang
    2. Input detail transaksi barang masuk & barang keluar, jika ada kode barang yg sama qty belum terakumulasi
    3. Input detail transaksi barang keluar belum ada validasi jika stok barang kosong atau kurang
    4. Report stok barang belum ada untuk menampilkan stok per barang, hanya per tanggal dokumen

Step 1 - Create Database

Create database melalui phpMyAdmin dengan nama database db_inventory

Step 2 - Copy Folder

Copy folder inventory ke folder htdocs

Step 3 - Setting file .env di folder project

Cari file .env di folder inventory, dan sesuaikan setting untuk koneksi dengan database yang sudah dibuat


Step 4 - Migration

Buka command prompt dan arahkan ke folder inventory
Jalankan migrasi tabel ke database dengan perintah php artisan migrate


Step 5 - Running

Masih di command prompt folder inventory, untuk menjalankannya ketik perintah php artisan serve


Selanjutnya buka browser dan bisa diakses dengan url http://127.0.0.1:8000
